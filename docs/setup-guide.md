## 1. Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js (v18 or later)**
  - [Download Node.js](https://nodejs.org/)
  - Verify: `node --version`, `npm --version`
- **Git**
  - [Download Git](https://git-scm.com/)
  - Verify: `git --version`
- **MetaMask Browser Extension**
  - [Download MetaMask](https://metamask.io/download/)
- **Code Editor**
  - Recommended: [VS Code](https://code.visualstudio.com/)

---

## 2. MetaMask Wallet Setup

1. **Install MetaMask**: Follow the link above and add it to your browser.
2. **Create Wallet**: Click "Create a new wallet", set a password, and **secure your recovery phrase**.
3. **Ready**: Network configuration happens in Step 7.

---

## 3. Clone & Install Dependencies

1. **Clone the repository**:
   ```bash
   git clone <your-repository-url>
   cd NyayaSetu
   ```
2. **Main App dependencies**: `npm install`
3. **Blockchain dependencies**: `cd blockchain && npm install && cd ..`
4. **Admin Panel dependencies**: `cd NyayaSetu-admin-master && npm install && cd ..`
5. **Generate Prisma client**: `npx prisma generate`

---

## 4. Database Setup (PostgreSQL via Neon)

1. **Create Neon account**: [neon.tech](https://neon.tech/)
2. **New Project**: Copy the connection string.
3. **Update `.env`**: Replace `DATABASE_URL` in `NyayaSetu/.env`.
4. **Push Schema**: `npx prisma db push`
5. **(Optional) View DB**: `npx prisma studio` (opens at `http://localhost:5555`)

---

## 5. Blockchain Setup (Hardhat Local Node)

1. **Start Node** (Terminal 1):
   ```bash
   cd blockchain
   npx hardhat node
   ```
2. **Keep Open**: This simulates your local Ethereum network.

---

## 6. Deploy Smart Contracts

(Terminal 2):
1. **Compile**: `cd blockchain && npx hardhat compile`
2. **Deploy**: `npx hardhat run scripts/deploy.cjs --network localhost`
3. **Addresses**: Automatically saved and injected into frontends.
4. **(Optional) DeadManSwitch**: `npx hardhat run scripts/deploy-deadman.cjs --network localhost`
5. **(Optional) Register Agency**: `npx hardhat run scripts/register-test-agency.cjs --network localhost`
6. **(Optional) Seed Data**: `npx hardhat run scripts/seed-cases.cjs --network localhost`

---

   - **RPC URL**: `http://127.0.0.1:8545`

## 9. Configure Environment Variables

1. **Check `.env.local`**: Auto-generated by deploy script.
2. **Main `.env`**: Set `PINATA_JWT`, `NEXT_PUBLIC_PINATA_GATEWAY`, and `DATABASE_URL`.

## 10. Start the Main Application

(Terminal 3):
```bash
cd NyayaSetu
npm run dev
```
Available at: `http://localhost:3000`

---

## 11. Start the Admin Panel

(Terminal 4):
```bash
cd NyayaSetu-admin-master
npm run dev -- -p 3001
```
Available at: `http://localhost:3001`

## 13. Quick-Start Commands (TL;DR)

1. **Terminal 1**: `cd blockchain && npx hardhat node`
2. **Terminal 2**: `cd blockchain && npx hardhat run scripts/deploy.cjs --network localhost`
3. **Terminal 3**: `npm run dev` (root)
4. **Terminal 4**: `cd NyayaSetu-admin-master && npm run dev -- -p 3001`

---

## 14. Troubleshooting

- **Hardhat Errors**: Ensure node is running; `npm install` in `blockchain`.
- **MetaMask Nonce**: Clear activity tab data in Advanced Settings.
- **Contract Errors**: Redeploy and restart Next.js server.
- **Database Errors**: `npx prisma db push`.
